# 青溪國小創造力資優班網站

這是一個以靜態網頁製作的班級成果展示網站，主題為「青溪國小創造力資優班 114 學年度」。網站內容包含班級介紹、課程架構、活動剪影、公告資訊與聯絡頁面，適合作為學校班級成果展示與對外資訊發布使用。

## 專案特色

- 使用 HTML 與 CSS 製作，結構簡單、容易維護
- 採用多頁式網站設計，方便分類呈現不同內容
- 以圖片與視覺版面為主，強調學習成果展示
- 適合作為 GitHub Pages 或一般靜態主機部署

## 網站頁面

- `index.html`：首頁
- `about.html`：班級介紹
- `curriculum.html`：課程架構
- `highlights.html`：活動剪影
- `notices.html`：公告資訊
- `contact.html`：聯絡我們

## 檔案結構

```text
tge-new-site/
├─ assets/
├─ about.html
├─ contact.html
├─ curriculum.html
├─ highlights.html
├─ index.html
├─ notices.html
└─ styles.css
```

## 使用方式

1. 下載或複製此專案。
2. 直接以瀏覽器開啟 `index.html`。
3. 若要發布到網路，可部署到 GitHub Pages、Netlify 或其他靜態網站服務。

## 適用情境

- 學校班級成果網站
- 課程活動展示頁面
- 校內專題或成果發表網站

## 技術說明

- HTML5
- CSS3
- 靜態資產管理方式為本地 `assets/` 資料夾引用

## 備註

本專案目前為靜態網站版本，若後續需要加入最新消息管理、表單串接或後台功能，可再延伸整合 JavaScript、表單服務或 CMS。
